# Movie Recommendation system

This Movie Recommendation System project leverages TMDB movie data to deliver personalized movie suggestions. Users can simply choose a movie from the dropdown box, triggering the system to showcase the top 5 most related movies along with their posters and names. The intuitive interface and dynamic dropdown make it easy to explore diverse movie options. Built with Streamlit and powered by the TMDB API, this recommendation system provides a visually engaging experience, allowing users to discover new and exciting films.
